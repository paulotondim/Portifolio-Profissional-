# Paulo Tondim - Portfólio Pessoal

Este projeto foi desenvolvido como atividade N3 da disciplina de Frontend Design Engineering, do curso de Análise e Desenvolvimento de Sistemas da UNIFASIPE.

## Sobre o projeto

A proposta deste trabalho foi criar uma Landing Page de apresentação pessoal utilizando HTML, CSS e JavaScript.

A página contém informações sobre mim, minhas habilidades, formas de contato e integrações com a BrasilAPI para consulta de CEP, CNPJ e feriados nacionais.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* BrasilAPI
* GitHub
* Vercel

## Funcionalidades

* Apresentação pessoal
* Links para redes sociais
* Consulta de CEP
* Consulta de CNPJ
* Consulta de feriados nacionais
* Layout responsivo para diferentes dispositivos

## Estrutura do projeto

```text
/
├── index.html
├── style.css

```

## Como executar

1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Utilize as funcionalidades disponíveis na página.
